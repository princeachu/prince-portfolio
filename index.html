Project Descriptions

EventScope is a collaborative full stack web application that allows the user to find events in their area. The frontend was designed using HTML5, CSS3 and Bootstrap. The backend was written utilizing JavaScript, Google API and Ticketmaster API.

    • BestPress is a collaborative full stack web application that allows users to create an account with a username and password and make posts interacting with other users. The frontend was designed using React and Semantics. The backend was written utilizing Node.JS, Mongoose, Express, Passport, Bcryptjs, Axios, and Cloudinary.

    • Note taker is a full stack application that can be used to post, save, and delete notes. The application's backend is managed through node.js and the server is set up using express. The frontend display was created using html and bootstrap css. The users notes are posted, saved and deleted through JQuery, AJAX and JSON.

    • Employee tracker is a terminal based application that allows a user to manage an employee database created using MySQL. Utilizing the application, the user will be able to add and remove an employee's information from the system. One can add what department they work in, their employee role, salary, as well as their role and department Id. This was accomplished by utilizing MySQL to store the data, node.js and inquirer to prompt the user on what information they would like to add, remove and view.
